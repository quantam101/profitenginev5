# ProfitEngine v5

Enterprise command center for local-first automation, declarative governance, health checks, and production release gates.

## Validate

```bash
npm ci
pip install -r requirements.txt
npm run ci:all
```

## Operate

Review `eaos.config.yaml`, `agents/registry.yaml`, `connectors/registry.yaml`, and `observability/slo.yaml` before release.
