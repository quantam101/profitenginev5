# Connector Access Checklist

## Free core

- local files
- local browser automation
- n8n local
- local model runtime
- local vector cache

## Free external

- GitHub read-only
- Cloudflare free DNS/proxy
- Google Search Console
- optional free-tier analytics

## Disabled paid adapters

- OpenAI API
- Anthropic API
- Perplexity API
- Devin
- Manus
- OpusClip Pro
- Zapier paid
- Stripe mutation

## Consent checkpoints

Human approval required before sending, posting, paying, merging, deploying, deleting, or changing account settings.
